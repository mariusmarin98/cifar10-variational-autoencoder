# Variational Autoencoder on CIFAR-10

This repository contains a Computer Vision coursework project focused on implementing and training a Variational Autoencoder (VAE) on the CIFAR-10 image dataset.

## Overview

The project explores representation learning using a variational autoencoder. The model learns a compressed latent representation of CIFAR-10 images and reconstructs images from this latent space.

## Objective

The main objectives of the project were to:

- implement and train a variational autoencoder for image data;
- explore latent representations learned from CIFAR-10 images;
- visualize reconstructed images generated by the model;
- analyze how image classes are represented in a lower-dimensional latent space.

## Dataset

The project uses the CIFAR-10 dataset, which contains 60,000 color images across 10 object classes.

## Methods

The project includes:

- image preprocessing for CIFAR-10;
- encoder and decoder architecture for a variational autoencoder;
- latent space sampling using the reparameterization trick;
- reconstruction loss and KL-divergence loss;
- training and evaluation of image reconstructions;
- visualization of 2D/3D latent representations.
